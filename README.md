## Bindings
| Binding | Function |
|---------|----------|
| <kbd>Ctrl</kbd> + <kbd>\\</kbd> | Open file explorer |
| <kbd>F8</kbd> | Change to dark/light theme |
| <kbd>Alt</kbd> + <kbd>j</kbd> | Switch to previous tab |
| <kbd>Alt</kbd> + <kbd>k</kbd> | Switch to next tab |
